# ManyIVsNets 0.1.0

## New Features

* Initial release implementing complete EPC analysis from scratch
* 24 different instrumental variable approaches
* Transfer entropy causal discovery with RTransferEntropy
* State-of-the-art alternative instruments (Spatial, Bartik, Migration, etc.)
* 7 publication-quality network visualizations
* Complete analysis of 49 countries (1991-2021)

## Instruments Implemented

* Geographic isolation instruments (proven strongest)
* Technology adoption instruments (internet, mobile, telecom)
* Migration network instruments (diaspora, language advantages)
* Geopolitical instruments (post-communist, NATO, EU membership)
* Financial development instruments (market maturity, banking)
* Natural risk instruments (seismic, volcanic, climate volatility)
* Transfer entropy-based instruments (network centralities)
* Alternative SOTA instruments (spatial lag, Bartik, shift-share, gravity)

## Key Results

* 21 out of 24 instruments show strong performance (F > 10)
* Best performing: Judge Historical SOTA (F = 7,155.39)
* Main finding: 1% ↑ unemployment → 0.071% ↓ CO2 emissions
* Transfer entropy network density: 0.095
* Country network density: 0.25
